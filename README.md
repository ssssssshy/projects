# Machine Learning Projects 🚀

Коллекция проектов по машинному обучению, реализованных в Jupyter Notebooks.

## 📋 Содержание проектов

### **Классификация и Регрессия**
- **logregression.ipynb** - Логистическая регрессия для задач классификации
- **linearregression_first.ipynb** - Линейная регрессия для прогнозирования
- **SVC.ipynb** - Метод опорных векторов (Support Vector Classification)
- **knn.ipynb** - K-Nearest Neighbors (метод k-ближайших соседей)

### **Кластеризация**
- **Kmeans.ipynb** - K-means кластеризация
- **DBSCAN.ipynb** - Density-Based Spatial Clustering

### **Снижение размерности**
- **PCA.ipynb** - Principal Component Analysis (метод главных компонент)

### **Прикладные проекты**
- **BestModel(lung_disease_data).ipynb** - Лучшая модель для диагностики заболеваний легких
- **lung_disease_dataV2.ipynb** - Анализ данных о заболеваниях легких (версия 2)
- **Workout & Fitness Tracker Dataset.ipynb** - Анализ фитнес-трекера и тренировок
- **a large fadango visualization project.ipynb** - Визуализация данных Fandango

## 🛠 Технологии и библиотеки

- Jupyter Notebook
- scikit-learn
- pandas

- seaborn
